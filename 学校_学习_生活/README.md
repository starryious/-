# 学校& 学习 & 生活

* 大一入学不必带太多东西, 有需要的东西列个清单, 可以上网购买, 也可以到时候一次性从家中全部寄过来.
* 选课时可以参考一下学长学姐的意见. 好老师和好课很重要.. 
* 11号宿舍楼免费热水免费洗衣服(bug). 
* 校医院报销带身份证, 不带身份证用电子会出现乱七八糟的问题
* 大一没想法前把 GPA 搞好很重要, 之后的选择更多

## 食堂美食

> 个人推荐, 主观成分较大

* 东一最北的包子, 粥, 水煮蛋,可以配上坚果和京东扶贫超市的水果(蓝莓, 草莓等)作为早餐

* 东一门口的现磨豆浆和红薯
* 西一轻食
* 西一豆浆, 红薯紫薯, 燕麦粥
